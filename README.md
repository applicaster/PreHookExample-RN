# React-Native Feed
## WORK IN PROGRESS

This project is the Feed built using react native.

# How to run the project:
- `npm install`
- `npm install -g react-native-cli`
- `react-native start`

You should now see the packager running on port 8081 and open your corresponding iOS/Android app.

# How to run tests:
- `npm test`
